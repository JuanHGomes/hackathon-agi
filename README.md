## Visão Geral  
Este projeto foi desenvolvido durante o hackathon do Agibank | Prosper Tech realizado em 07/11 ~ 08/11. O objetivo principal era criar uma solução inovadora para um sistema de gestão de tarefas e usuários

## Funcionalidades Principais  
- login e cadastro – com validação de e-mail e senha
- Criação e gerenciamento de usuário
- Criação, atribuição e acompanhamento de de Tarefas 
- Atualização e conclusão de tarefas
- Visualização e controle de status
- Geração de histórico
  
## Tecnologias Utilizadas  
- Linguagem principal: Java  
- Frameworks / ferramentas: Spring Boot e Spring Security, MySQL
- Build: Gradle
- Apresentação: Swagger
